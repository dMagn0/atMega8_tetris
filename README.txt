*atMega16*

PA0(ADC0) -> joystick x_axis
PA1(ADC1) -> joystick y_axis
PA2 -> joystick button

PB4(SS) -> MAX7219 LOAD
PB5(MOSI) -> MAX7219 DIN
PB6(MISO) <- None
PB7(SCK) -> MAX7219 CLK



*atMega8*

PC0(ADC0) -> joystick x_axis  
PC1(ADC1) -> joystick y_axis
PC2 -> joystick button
precisa inverter:
PC2(ADC2) -> joystick x_axis  
PC1(ADC1) -> joystick y_axis
PC0 -> joystick button



PB2(SS) -> MAX7219 LOAD
PB3(MOSI) -> MAX7219 DIN
PB4(MISO) <- None
PB5(SCK) -> MAX7219 CLK
